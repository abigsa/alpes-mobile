# Alpes Mobile

Proyecto fullstack con backend Node.js/Express/Oracle y frontend Flutter.

## Estructura

```
alpes-mobile/
├── backend/          → API REST Node.js + Express + OracleDB
└── mobile/           → App Flutter (Android/iOS)
```

## Backend

```bash
cd backend
npm install
cp .env.example .env   # configurar variables
npm run dev
```

## Mobile

```bash
cd mobile
flutter pub get
flutter run
```
